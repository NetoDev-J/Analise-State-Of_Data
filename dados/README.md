# Dados — State of Data Brazil 2024–2025

## Como baixar a base

1. Acesse: [kaggle.com/datasets/datahackers/state-of-data-brazil-20242025](https://www.kaggle.com/datasets/datahackers/state-of-data-brazil-20242025)
2. Faça login ou crie uma conta gratuita no Kaggle
3. Clique em Download no canto superior direito da página do dataset
4. Extraia o arquivo `.zip` — o CSV principal é `state_of_data_brazil_2024_2025.csv`
5. Mova o arquivo para esta pasta (`dados/`)

## Como reproduzir a análise

Com o CSV na pasta `dados/`, abra o notebook em:

```
notebook/JoseAlves-EnzoSantos-analise.ipynb
```

Certifique-se de que as seguintes bibliotecas estão instaladas:

```bash
pip install pandas numpy matplotlib seaborn scipy
```
